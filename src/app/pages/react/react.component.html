React

React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.

  • Al igual que angular, se levanta con npm start en el localhost:3000
  • Se importan módulos que hay que instalar con npm install
  • Los recursos gráficos van en la carpeta public/
  • Los estilos css/scss van en style/
  • Una pantalla, por ejemplo home, es un archivo .js. En la parte de arriba van las importaciones, funciones, etc. En la parte de abajo, dentro de un return() va el código html.
  • Para estilar un html hay que importar (o sea vincular) el archivo scss a ese archivo .js: import '../styles/home.scss';
  • React, tiene una estructura de tipo árbol. Hay un "componente" inicial y se va abriendo en otros componentes que heredan.
